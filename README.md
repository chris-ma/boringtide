# Boringtide Fishing

`Boringtide Fishing` is a small browser prototype for a top-down, arcade-style shore fishing game.

## What It Includes

- Tap-to-cast controls built for mobile-friendly play
- Automatic retrieve with optional twitch and fast-burn inputs
- Randomized Australian pelagic catches with species-based weight ranges
- Short tide-session timer with rising shark danger
- Arcade catch popups and a shark-loss game-over screen

## Running It

Open [index.html](./index.html) in a browser.

## Files

- `index.html` contains the UI shell and overlays
- `styles.css` contains the arcade presentation and responsive layout
- `game.js` contains the game loop, fish system, lure behavior, and session logic
